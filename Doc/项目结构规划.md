# ATE Pattern Card 项目结构规划

## 一、项目概述

自制ATE系统Pattern卡，参照NI PXIe-6571规格设计，核心参数：
- **通道数**：16路DUT (基于ADATE305双通道DCL芯片，需8片)
- **速率**：100MHz / 每路
- **接口**：PXIe
- **参考设计**：NI PXIe-6571

## 二、系统架构（双板方案）

系统由两块板卡通过Board-to-Board连接器组成：

### 板卡1：Ku035_Board（FPGA主控板）
- **FPGA**: Xilinx XCKU035 (UltraScale)
- **接口**: PXIe背板连接器 (PCIe Gen2x4)
- **存储**: DDR3 + QSPI Flash
- **功能**:
  - PXIe总线通信 (PCIe)
  - Pattern存储与控制逻辑 (Large Vector Memory)
  - 时序引擎 (Timing Engine, 100MHz vector rate)
  - SPI控制16片ADATE305
  - ADC数据采集控制
  - 时钟管理 (PXIe_CLK100)
  - Trigger总线接口
  - OVD (过压检测) 信号采集

### 板卡2：DLC_Board（DCL模拟前端板）
- **DCL芯片**: ADATE305BSVZ × 8片 (每片2通道，共16 site)
- **ADC**: ADS7959 × 2片 (采集MEASOUT/DAC16_MON)
- **连接器**: BTB连接器 × 4组 (J2A~J2J，连接到Ku035板)
- **14层PCB** (信号完整性要求高)
- **功能**:
  - 16通道Pin Electronics (Driver/Comparator/Active Load/PMU)
  - 模拟电平设置 (14-bit DAC, 16-bit DAC)
  - 过压检测 (OVD)
  - 参考电压 (VREF 5V, ADR431)
  - 多路电源 (+16.75V, +10V, -5.25V, +3.3V, +1.8V, +5V参考)

## 六、校准系统设计（参照NI PXIe-6571 Calibration Procedure）

### 6.1 校准概述

校准分为三个层次：
1. **Self-Calibration（自校准）**：上电预热30分钟后执行，测量板载参考并调整内部校准常数
2. **Verification（验证）**：使用外部标准仪器验证各项指标是否在规格范围内
3. **Adjustment（调整）**：若验证不通过，通过外部标准调整校准常数并存入EEPROM

推荐校准周期：**1年**

### 6.2 校准项目

| 校准项 | 测试内容 | As-Found限值 | 所需外部仪器 |
|--------|----------|-------------|-------------|
| **Driver电压精度** | VIH/VIL/VTERM在-2V~6V范围内4个测试点 | ±15 mV | DMM (6½位) |
| **Comparator阈值精度** | VOH/VOL在-1.5V~5.8V范围内4个测试点 | ±25 mV | DMM + SMU |
| **PPMU Force电压精度** | -2V~6V范围内4个测试点 | ±15 mV (1MΩ负载) | DMM (6½位) |
| **PPMU Measure电压精度** | -2V~6V范围内4个测试点 | ±5 mV | DMM (6½位) |
| **PPMU Force电流精度** | 5档量程(±2μA~±32mA)，Zone1区域 | ±1% of range | SMU |
| **PPMU Measure电流精度** | 5档量程，Zone1 & Zone2 | Zone1: ±1%, Zone2: ±1.5% | SMU |
| **Force电压象限(Zone2)** | 不同象限下Force电压精度 | ±15 mV | SMU |

```

    │
    ▼
Self-Calibration（自校准）
    │  测量板载0V和5V参考电压
    │  测量板载50Ω参考电阻
    │  更新内部校准常数
    │
    ▼
┌────────────────────────────────────────┐
│      Verification（外部验证）            │
│                                        │
│  1. Driver电压精度验证                   │
│     - 对每通道：设VIH/VIL/VTERM         │
│     - DMM测量DUT输出电压                 │
│     - 误差 = V_setpoint - V_measured    │
│     - 判定：|误差| ≤ 15mV → PASS        │
│                                        │
│  2. Comparator阈值精度验证               │
│     - SMU向DUT通道施加扫描电压            │
│     - 二分法搜索比较器翻转阈值            │
│     - DMM测量翻转点电压                  │
│     - 判定：|误差| ≤ 25mV → PASS        │
│                                        │
│  3. PPMU Force/Measure电压验证           │
│     - PPMU输出电压，DMM测量              │
│     - Force误差 ≤ ±15mV               │
│     - Measure误差 ≤ ±5mV              │
│                                        │
│  4. PPMU Force/Measure电流验证           │
│     - SMU设定电压，PPMU输出电流           │
│     - SMU测量实际电流（取反）             │
│     - 5档量程：±2μA/±32μA/±128μA/      │
│       ±2mA/±32mA                      │
│     - Zone1: ±1% of range → PASS      │
│     - Zone2: ±1.5% of range → PASS    │
│                                        │
│  5. Force电压象限验证（Zone2）            │
│     - SMU源电流，PPMU Force电压           │
│     - 验证四象限工作精度                  │
└────────────────────────────────────────┘
    │
    ▼ (任一项FAIL)
┌────────────────────────────────────────┐
│      Adjustment（外部调整）              │
│                                        │
│  1. 调整电压和电阻参考                    │
│     - 校准路径：Voltage Reference        │
│     - DMM测0V参考 → 存为M0              │
│     - DMM测5V参考 → 存为M1              │
│     - 写入DUT校准常数                    │
│     - 校准路径：Resistor Reference       │
│     - SMU源8组电流(±0.5mA~±50mA)       │
│     - DMM测两端电压差 → 计算电阻值        │
│     - 写入50Ω参考校准常数                │
│                                        │
│  2. 调整Force/Measure电流增益/偏移        │
│     - 对5档量程分别校准                   │
│     - SMU设定电压，PPMU源电流             │
│     - 记录SMU测量电流和PPMU测量电流       │
│     - 计算增益/偏移校准常数               │
│     - 写入DUT EEPROM                    │
└────────────────────────────────────────┘
    │
    ▼
Reverification（再验证，使用As-Left限值）
    │
    ▼
```

### 6.5 本项目校准设计要点

**硬件设计要求**：
- DLC_Board需预留 **CAL_FORCE / CAL_SENSE / CAL_MEASURE / CAL_GND** 校准引脚（通过VHDCI连接器引出）
- 板载 **ADR431** 精密5V参考需作为校准基准
- ADATE305 内部50Ω精密电阻可作为电阻校准参考
- 需支持 **每通道独立校准**（通过SPI配置+开关矩阵切换）
- 校准常数存储在 **FPGA端的EEPROM/Flash** 中（非ADATE305内部）

**FPGA设计要求**：
- 校准常数存储模块（Flash/EEPROM读写）
- Self-Calibration序列控制（自动测量板载参考并更新常数）
- 校准模式控制（允许软件直接控制单通道PPMU/Driver状态）
- DAC校准补偿计算（对14-bit/16-bit DAC的增益/偏移校正）

**软件/API设计要求**：
- `SelfCalibrate()` — 触发自校准
- `ConfigureCalibrationReference(path, reference)` — 配置校准参考路径
- `AdjustCalibrationReference(path, reference, value)` — 写入校准调整值
- `InitExternalCalibrationSession(password)` — 初始化外部校准会话
- `CloseExternalCalibrationSession()` — 关闭校准会话并保存
- `GetLastCalibrationInfo()` — 读取上次校准日期/温度

### 6.6 校准精度预算

| 指标 | NI PXIe-6571目标 | ADATE305芯片能力 | 本项目预估 |
|------|-----------------|------------------|-----------|
| Driver电压精度 | ±15 mV | ±10 mV INL (校准后) | ±15 mV (含PCB/连接器误差) |
| Comparator精度 | ±25 mV | — | ±25 mV |
| PPMU Force电压 | ±15 mV | ±1.5 mV INL (16-bit DAC) | ±15 mV |
| PPMU Measure电压 | ±5 mV | 取决于外部ADC (ADS7959) | ±10 mV (12-bit ADC限制) |
| PPMU Force电流 | ±1% of range | 5档: 32mA~2μA | ±1% |
| PPMU Measure电流 | ±1%/±1.5% | 取决于外部ADC | ±2% (待验证) |

**注意**：NI PXIe-6571使用专用ASIC（16-bit ADC内置），PPMU Measure精度较高。本项目使用ADS7959（12-bit SAR ADC），Measure电压/电流精度会受限，可能需要过采样或升级为更高精度ADC。

## 七、电源架构总结

### Ku035_Board 电源树（从PXIe +12V/+3.3V派生）
```
PXIe +12V ──┬── TPS53513 ──→ VCCINT 1.0V @8A (FPGA Core)
            ├── TPS53513 ──→ MGTAVCC 1.0V @8A (FPGA MGT)
            ├── TPS568230 ──→ 1.5V @3A (DDR3)
            ├── LM51551 ──→ 1.8V @3A (FPGA IO)
            ├── LM51551 ──→ MGTAVTT 1.2V @3A (FPGA MGT)
            ├── SPX3819 ──→ 2.5V @100mA (FPGA IO)
            ├── TPS563240 ──→ VDD 10V @5A (ADATE305)
            ├── TPS62823 ──→ VPLUS 16.75V @3A (ADATE305)
            ├── Isolated ──→ VSS -5.25V (ADATE305)
            └── TPS62823 ──→ OPA VDD +5V @2A

PXIe +3.3V ──→ 3.3V logic
```

### DLC_Board 电源（从Ku035板通过BTB连接器供电）
```
VPLUS 16.75V ──→ ADATE305 VPLUS (120-150mA/chip × 8 = ~1.2A)
VDD 10V ──────→ ADATE305 VDD (160mA/chip × 8 = ~1.3A)
VSS -5.25V ───→ ADATE305 VSS (183mA/chip × 8 = ~1.5A)
VDD 3.3V ─────→ ADATE305 VCC + COMP_VTT
VDD 1.8V ─────→ ADATE305 逻辑
VREF 5V ──────→ ADATE305 VREF (ADR431 产生)
```
